# Fondation XP & niveau (sous-projet A) — Plan d'implémentation

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ajouter une couche d'XP par perso qui, à un seuil, fait monter automatiquement le `state/level` déjà en place (lequel débloque comps/runes/passif).

**Architecture:** Logique pure testée (`xpToNext`/`applyXp` dans `game-logic.js`) + orchestrateur `addXp(charId, gain)` (data-state, lecture `getSnapshot` → écriture patch `{level, xp}` + `pushLog` au level-up) + composant lecture seule `XpBar` affiché sur fiche / Progression / cartes MJ, avec un contrôle « +XP » ad-hoc côté MJ.

**Tech Stack:** React 18 + Babel standalone (CDN, zéro build), Firebase RTDB compat, `node --test`.

## Global Constraints

- **Zéro build** : chaque `.jsx`/`.js` définit ses symboles puis `Object.assign(window, {...})` ; les autres fichiers y accèdent par référence nue (résolue via `window`). Ne jamais ajouter d'`import`/`require` dans les fichiers navigateur.
- **`game-logic.js`** est UMD pur (aucune dépendance React/DOM/Firebase) : testable en Node. Toute nouvelle fonction pure va là et est ajoutée au `return { … }` de la factory.
- **Tests** : seule la logique pure de `game-logic.js` a des tests unitaires (`node --test test/game-logic.test.js`). Les `.jsx` se vérifient par `npx esbuild <fichier> >/dev/null` (syntaxe) ; aucun framework de test JSX.
- **Toasts** : `toast` n'est PAS global — c'est `const toast = useToast();` (hook). Seuls les composants React peuvent toaster. `pushLog(text, kind)` (global) journalise (`kind ∈ 'gold'|'buff'|'debuff'`, seul `<b>` est rendu).
- **Aucune nouvelle règle RTDB** : `xp` vit dans `characters/$charId/state`, déjà couvert (joueur = sa fiche R/W, staff = tout). **Ne pas toucher `database.rules.json`.**
- **`state/level` reste l'autorité du niveau** ; `xp` = progression DANS le niveau courant (0 ≤ xp < seuil). Le niveau effectif se lit `state.level ?? char.level`.
- Commits fréquents, un par tâche.

```js
  /* XP & niveau : courbe générique (à remplacer plus tard) + application d'un gain.
     xp = progression DANS le niveau courant ; le surplus reporte en cascade. */
  function xpToNext(level) { return 100 * Math.max(1, level | 0); }
  function applyXp(level, xp, gain) {
    level = Math.max(1, level | 0);
    xp = Math.max(0, xp | 0) + Math.max(0, gain | 0);
    let levelsGained = 0;
    while (xp >= xpToNext(level)) { xp -= xpToNext(level); level += 1; levelsGained += 1; }
    return { level, xp, levelsGained };
  }
```

**Interfaces:**
- Consumes: `applyXp` (Task 1, global) ; `charPath`, `window.RTDB.getSnapshot`/`updatePath`, `pushLog`, `CHARACTERS` (globals existants).
- Produces: `async addXp(charId, gain) -> { level, xp, levelsGained }` — lit l'état du perso ciblé, applique le gain, écrit `{level, xp}`, journalise la montée. `gain ≤ 0` → no-op (`{ level:null, xp:null, levelsGained:0 }`).

- [ ] **Step 1 : Implémenter `addXp`** — dans `data-state.jsx`, juste après la fonction `useCombatLog` (après sa `}` de fermeture, ~ligne 139), insérer :

```js
/* Don d'XP (orchestrateur, écriture staff). Lit l'état du perso ciblé, applique le
   gain via applyXp (montée auto + report du surplus), écrit { level, xp }. Journalise
   la montée (pushLog). Retourne le résultat pour que l'appelant puisse toaster. */
async function addXp(charId, gain) {
  gain = Math.max(0, gain | 0);
  if (!gain) return { level: null, xp: null, levelsGained: 0 };
  const c = CHARACTERS.find(x => x.id === charId);
  const p = charPath(charId);
  const st = (await window.RTDB.getSnapshot(p)) || {};
  const curLevel = (st.level != null ? st.level : (c ? c.level : 1)) || 1;
  const curXp = Math.max(0, st.xp | 0);
  const res = applyXp(curLevel, curXp, gain);
  window.RTDB.updatePath(p, { level: res.level, xp: res.xp });
  if (res.levelsGained > 0) pushLog(`<b>${c ? c.name : charId}</b> passe niveau <b>${res.level}</b> !`, 'buff');
  return res;
}
```

```jsx
/* --- Barre d'XP (lecture seule) : progression DANS le niveau courant. --- */
function XpBar({ level, xp }) {
  const lvl = Math.max(1, level | 0);
  const need = xpToNext(lvl);
  const cur = Math.max(0, xp | 0);
  const pct = need > 0 ? Math.max(0, Math.min(100, (cur / need) * 100)) : 0;
  return (
    <div className="col gap-1">
      <div className="row" style={{ justifyContent:'space-between', alignItems:'baseline' }}>
        <span className="overline">Niveau {lvl}</span>
        <span className="mono faint" style={{ fontSize:11 }}>{cur} / {need} XP</span>
      </div>
      <div className="bar">
        <div className="fill" style={{ width: pct + '%', background:'var(--gold-bright)' }}></div>
      </div>
    </div>
  );
}
```
